Программа курса

4 недели интенсивного обучения с нуля до профессионального уровня

Неделя 1: Основы HTML

Структура HTML-документа, работа с текстом, списки, ссылки, изображения. Семантическая верстка. Формы и элементы форм. Основы доступности.

Неделя 2: Основы CSS

Селекторы, каскадность, наследование. Работа с текстом, цветами, фонами. Блочная модель, Flexbox. Позиционирование элементов. Основы адаптивности.

Неделя 3: Продвинутый CSS

Grid Layout, анимации и переходы. Переменные CSS. Методологии именования (БЭМ). Препроцессоры SASS/SCSS. Адаптивная и мобильная верстка.

Неделя 4: Финальный проект

Верстка многостраничного сайта. Оптимизация производительности. Кроссбраузерная верстка. Размещение проекта в интернете. Сборка проекта.
